Afficher les visages dans les messages :

Ce tutorial va vous expliquer comment afficher des visages dans vos messages comme par exemple le jeux Golden Sun.

Ajouter l’image de celui qui parle :

Vous devez d’abord ajouter l’image qui représente celui qui parle je vous conseil de faire avec les avatars , pour ajouter l’image procédez comme ceci :

Ouvrez le répertoire de votre jeu en faisant dans RPG Maker Xp : Jeu-> Ouvrire le répertoire du jeu

Puis ouvrez le dossier Graphics\Pictures\ et mettez dedans les images que vous vouler afficher dans vos messages.

Voila faites comme ca avec toutes vos images a afficher. Ensuite dans RPG Maker Xp  creéz l’évènement suivant :

Explications :

  • On créé d’abord une boucle pour que l’image et le message s’affiche en même temps (ne vous demandez pas pourquoi si vous y connaissez rien a la programmation).
  • On affiche l’image grâce a la commande Afficher une image  q’on appelleras Image 1. Puis ont définie l’emplacement  de l’image avec  X Y (oui comme dans les maths) X=20 et Y=150 dans mon exemple en partant du premier pixel en haut a gauche.
  • Ensuite ont affiche le message .
  • Ont supprime l’image sinon elle va rester tout le long du jeu.
  • Et enfin ont sors de la boucle parceque sinon l’évènement va se répéter éternellement.
